# RunningFit V4.5 App Spec

복구된 핵심 기능:
- 오늘 러닝 추천
- 내 신발 관리
- 내 프로필
- 대회 등록 및 D-day
- 오늘 훈련 프로그램 추천
- 러닝 기록 추가
- 캡처 업로드 미리보기
- 입문 추천
- 정밀 추천
- 러닝화 DB 검색/자동완성/초성검색/TOP10 유지

데이터 저장:
- 현재 버전은 브라우저 localStorage에 저장한다.
- 서버/로그인/회원가입은 다음 단계에서 연결한다.

캡처 등록:
- V4.5는 이미지 미리보기 + 직접 기록 입력 방식.
- OCR 자동 분석은 다음 단계.
